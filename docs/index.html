The problem

Autonomous agents, autonomous losses.

A token can pass every surface check. The trap only reveals itself when an agent tries to exit.

▲An agent can't pause to feel suspicious.

∞Every agent re-checking every token wastes the network's effort.

Without shared attestations, each agent repeats the same screening work independently.

How it works

Trust as shared infrastructure

Screen

An agent gathers risk intel — GoPlus security data plus on-chain heuristics — and computes a deterministic 0–100 score.

Attest

The verdict is written to the registry on Pharos: ALLOW / WARN / BLOCK, with reasons. Permanent, public, no admin key.

Reuse

Every other agent reads the attestation in one free view call instead of re-screening. Unknown ≠ safe. Screenings can carry PHRS bounties, auto-paid to attesters.
